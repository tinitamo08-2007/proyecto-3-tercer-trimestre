# Base de datos — Sistema de Gestión de Granja

Esta carpeta contiene todos los entregables relacionados al diseño y la implementación de la base de datos del proyecto.

## Estructura

| Carpeta | Contenido | Entregable del enunciado |
|---|---|---|
| `01-diseno-conceptual/` | Modelo Entidad/Relación | Punto 1 |
| `02-diseno-logico/` | Modelo relacional (paso de E/R a tablas) | Punto 2 |
| `03-normalizacion/` | Análisis de normalización 1FN, 2FN, 3FN | Punto 3 |
| `04-script-sql/` | Scripts SQL de creación de la base | Punto 4 |
| `05-consultas/` | Consultas usadas por Java y la versión Web | Punto 5 |
| `06-respaldos-ejemplo/` | Muestras de respaldos en CSV | Extras |

## Orden de lectura recomendado

Para entender el proceso de diseño completo, leer las carpetas en orden numérico. Cada carpeta contiene su propio README con la justificación de las decisiones tomadas en esa etapa.

## Tecnología

- Motor: **MySQL 8.0+**
- Codificación: `utf8mb4_unicode_ci`
